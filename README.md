#Stacks: 

- Node.js (Express 5)

- Google Gemini API (@google/genai)

- Vanilla JS frontend

- Postman API

#Local Setup:

1. Create .env file:
```
GEMINI_API_KEY=your_api_key_here
GEMINI_MODEL=gemini-2.5-flash
```

2. Run in your terminal:
```bash
npm install
npm install @google/generative-ai
node index.js
```
3. Click the local host link to see the webpage
